# gonta-app

`gonta-app` is a slack app written in golang.

# HowToDev

## Environments

| Name | Description |
|:-----|:-----------:|
| GO111MODULE | on |
| GCP\_PROJECT | `your gcp project` |
| GCP\_SERVICE\_ACCOUNT | `your service account email` |

## Run

```
$ make run
```
